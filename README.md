# Oil Sales Analysis

This repository contains a data exploration and modeling notebook to analyze oil sales transactions.

Files:
- `oil_sales_analysis.ipynb` — Annotated notebook with EDA, data quality checks, two baseline models (Random Forest regression and classification), and business insights.
- `oil_sales_assignment_dataset.csv` — Source dataset used throughout the notebook.

Running locally (Windows / PowerShell):
1. Create a virtual environment (optional but recommended):

```powershell
python -m venv env;  
env\Scripts\Activate
```

2. Install dependencies:

```powershell
pip install -r requirements.txt
```

3. Start Jupyter (or open `oil_sales_analysis.ipynb` in VS Code):

```powershell
jupyter notebook
```
